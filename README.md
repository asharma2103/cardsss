# MadeWithEfforts ❤️

A beautiful, interactive, and premium memory-sharing web app built with pure love and clean code. Designed to celebrate special moments with animations, interactive elements, and a smooth user experience.

## ✨ Features

* **Stage 1: Cinematic Intro** – Elegant animations with an interactive floating bokeh and heartbeat background.
* **Stage 2: Drag to Reveal Magic** – A smooth custom heart-slider that fades out a romantic note and unfolds a beautiful full-screen memory video.
* **Stage 3: Interactive Memory Game** – 6 premium memory cards that stack dynamically, flip seamlessly on click with a magical burst effect, and support free-hand dragging and shuffling.
* **Fully Responsive** – Tailored for a perfect experience across laptops, tablets, and smartphones.

## 🛠️ Tech Stack

* **HTML5** – Structured with Semantic layout and semantic video features.
* **CSS3** – Custom keyframes for physics-based floating hearts, bokeh effects, and responsive layout grids.
* **Vanilla JavaScript** – Handles interactive custom drag events (touch & mouse), shuffling logic, and stack rendering.
